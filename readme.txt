Mob Quotes is an admin panel quotes plugin inspired by famous gangsters of the 20th Century.

== Description ==


= Plugin URL =
 https://alanhylands.com/wp-plugins-mob-quotes

"Inspired" by the most famous Wordpress plugin of all, Hello Dolly by Matt Mullenweg, "Mob Quotes" is a quotes plugin inspired by famous and infamous mobsters of the 20th century. From Gambino to Gotti, Lansky to Trafficante, Mob Quotes will display random quotes from the top Mafia crime kingpins in your Wordpress admin dashboard. Building on the fine basis of Hello Dolly, Mob Quotes is a plugin with an edgier undertone. Capische?






== Installation ==



To install the plugin just follow these steps:



1. Download the plugin and unzip it.


2. Copy the mob-quotes folder into your plugins folder ( wp-content/plugins ).


3. Log-in to the WordPress administration panel and go to the Plugins page.


4. Find the Mob Quotes plugin and click on the Activate link.

or



1. Click on Plugins -> Add New in your Wordpress admin panel.



2. In the "Search plugins..." box, type Mob Quotes.



3. When the results come back, look for the one by Alan Hylands and click the Install Now button.



4. When it is installed, click on the Activate link.
